# Guía Rápida de Gestor Archivos

Esta guía rápida cubre las acciones más comunes.

## 1. Navegar y Abrir

- Pulsa carpetas para navegar.
- Pulsa un archivo para abrirlo.
- Usa el breadcrumb para volver a carpetas anteriores.

## 2. Buscar

- Pulsa el icono de búsqueda.
- Escribe parte del nombre del archivo o carpeta.
- Pulsa Cancelar para detener una búsqueda larga.

## 3. Seleccionar y Gestionar

- Mantén pulsado un elemento para entrar en modo selección.
- Acciones inferiores: Enviar, Abrir, Mover, Copiar, Eliminar, Renombrar (solo 1 elemento) e Info (solo 1 fichero).
- Usa Mover/Copiar, luego ve a la carpeta destino y pulsa Pegar.

## 4. Recientes

- Abre la pestaña Recientes para ver archivos abiertos recientemente.
- Mantén pulsado un elemento reciente para acciones rápidas.
- Limpiar recientes solo borra la lista, no tus archivos.

## 5. Papelera

- Eliminar envía elementos a la Papelera por defecto.
- Puedes restaurar elementos desde la Papelera.
- Vaciar Papelera elimina definitivamente.

## 6. Importar desde Compartir (Guardar aquí)

- Comparte archivos a Gestor Archivos desde otra app.
- En Gestor Archivos, navega a la carpeta destino.
- Pulsa Guardar aquí para importar archivos compartidos en cola.

## 7. Apps por Defecto

- Selecciona un solo fichero y pulsa Abrir para definir app por defecto.
- Las reglas se guardan por extensión de archivo.
