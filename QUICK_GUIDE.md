# File Manager Quick Guide

This quick guide covers the most common actions.

## 1. Browse and Open

- Tap folders to navigate.
- Tap a file to open it.
- Use the breadcrumb to go back to parent folders.

## 2. Search

- Tap the search icon.
- Type part of the file or folder name.
- Tap Cancel to stop a long search.

## 3. Select and Manage

- Long-press an item to enter selection mode.
- Bottom actions include: Send, Open, Move, Copy, Delete, Rename (single item), and Info (single file).
- Use Move/Copy, then navigate to a destination folder and tap Paste.

## 4. Recents

- Open the Recents tab to see recently opened files.
- Long-press a recent item for quick actions.
- Clearing recents removes only the list, not your files.

## 5. Trash

- Deleting sends items to Trash by default.
- You can restore items from Trash.
- Empty Trash deletes items permanently.

## 6. Share Import (Save Here)

- Share files to File Manager from another app.
- In File Manager, navigate to the destination folder.
- Tap Save here to import queued shared files.

## 7. Default Apps

- Select a single file and tap Open to set its default app.
- Default app rules are stored by file extension.
